Al momento de escribir la presente información, existen un total de 3 versiones distintas del programa Nginx, siendo estas Estable(Stable), de Desarrollo(Development), de Legado(Legacy), por lo que a continuación, procedere a hablar de forma resumida a cerca de cada una de las 3 anteriormente mencionadas.

•ESTABLE(STABLE): Es la más recomendada para utilizacion tando para principiantes como para el publico en general.

•DESARROLLO(DEVELOPMENT): Esta es la version que contiene las nuevas funciones y en la que los errores del programa son eliminados primero, pero tambien en la que hay mayor probabilidad de que aparezcan nuevos errores.

•LEGADO(LEGACY): Finalmente, esta ultima es la version que fue Estable hasta que la nueva version apareciece, se mantiene para eliminar problemas de seguridad mayormente.
